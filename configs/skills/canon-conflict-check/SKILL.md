# canon-conflict-check

## 目标

在世界资料中发现潜在冲突，给出最小可执行的修复建议。

## 输入范围

- 世界卡（world card）
- 世界规则（rules）
- 正典补充（canon addenda）
- 提案/编年史文本

## 检查步骤

1. 先提取事实主语：人物、势力、地点、时间、规则。
2. 比较同一主语的关键字段：时间线、身份、约束、资源、结果。
3. 标记冲突类型：
   - 时间冲突
   - 身份/关系冲突
   - 规则冲突
   - 事件因果冲突
4. 对每个冲突给出最小修复建议（优先改一处而不是大面积重写）。

## 输出格式

- 冲突摘要（按严重级别排序）
- 每条冲突对应的证据片段（简述，不大段引用）
- 每条冲突给 1-2 个可执行修复方案
