# 角色卡生成技能

## 技能说明

本技能用于从用户输入的任意格式文档中提取角色信息，转换为结构化表格。适用于所有类型世界的角色。

## 在本项目中的文件约定（Discord 世界系统）

当你在本仓库的“角色构建会话”（通常由 `/character create` 自动创建的话题）中使用本技能时：

- 必须先读取：`world/world-card.md` 与 `world/rules.md`（世界正典与硬规则，只读约束）
- 必须写入：`world/character-card.md`（本角色卡产物）
- 必须通过工具写入/编辑文件，禁止只在聊天里输出最终内容
- 禁止修改 `world/world-card.md` 与 `world/rules.md`（即使你改了也不会被保存）
- 回复中只做：变更摘要 + 3-5 个需要创作者补充的问题（缺什么问什么）

## 适用范围

| 类型   | 角色示例                                   |
| ------ | ------------------------------------------ |
| 日常类 | 学生、上班族、普通人、ABO角色、哨向角色    |
| 奇幻类 | 修仙者、魔法师、骑士、冒险者、轮回者       |
| 科幻类 | 赛博人、宇航员、变种人、义体人、星际舰长   |
| 恐怖类 | 调查员、猎人、幸存者、规则怪谈参与者       |
| 游戏类 | 训练师、决斗者、玩家、系统宿主、无限流玩家 |
| 魂系类 | 不死人、褪色者、灰烬、循环者               |
| 表里类 | 人格面具使用者、双面身份者                 |
| 机甲类 | 驾驶员、适格者、舰长                       |
| 探索类 | 迷宫探索者、爬塔者、地下城冒险者           |
| 混合类 | 都市异能者、机甲驾驶员                     |

## 使用规则

1. 所有模块均为可选，根据用户内容决定填充
2. 特殊机制模块根据世界观类型选用
3. 支持用户自定义字段
4. 与世界观卡联动，自动匹配机制
