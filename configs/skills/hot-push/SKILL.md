# hot-push

## 目标

用于系统定时推送场景：基于 TalesOfAI MCP 获取当天热点，输出简洁可读的摘要。

## 执行规则

1. 优先调用 TalesOfAI MCP 获取当天热点内容（热榜/热门话题/热门作品/热门角色等）。
2. 只保留高相关、可公开的信息，不输出内部信息。
3. 不得输出任何敏感信息（token/key/密码/内部链接等）。
4. 若数据源不可用或检索失败，直接说明失败原因，不要伪造结果。

## 输出格式（必须）

- 精选 5 条，每条 1-2 句。
- 末尾附 1 句总结。
- 不要输出冗长解释。
