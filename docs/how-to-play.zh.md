# 如何游玩（Discord 版）

这是一份给普通用户看的「上手指南」，目标只有一个：让你 **最快开始玩**，并在遇到问题时知道该用哪个指令解决。

> 你不需要记住所有指令。先按“快速开始”走一遍，剩下的用到再查。

## 你需要先理解的 3 个概念

1. **会话**：一段对话上下文。你可以同时开多个会话槽位（例如 `#0`、`#2`），互不干扰。
2. **角色**：你扮演的人物卡。你可以有很多个角色，随时切换。
3. **世界**：一套世界观与规则。你需要加入某个世界，才能在该世界里推进剧情、提交设定与任务。

## 快速开始（冒险者：3 分钟上手）

### 1）打开新手引导

在服务器任意可用频道输入：

```text
/onboard role:adventurer
```

系统会给你一个私密引导话题（线程）。**后续推荐都在这个话题里操作**，省得找频道、也不需要每句都 @机器人。

### 2）设置语言（建议先做）

```text
/language lang:zh
```

### 3）准备一个角色

如果你还没有角色，直接创建一个（默认私有）：

```text
/character create
```

如果你已经有角色：

```text
/character list
/character use character_id:<id>
```

> `use` 是“你的默认角色（全局）”。进入某个世界后，你还可以用 `act` 指定“这个世界里当前使用的角色”。

### 4）加入一个世界并开始玩

先找世界：

```text
/world list
```

加入世界（可选指定角色；不填则用你当前默认角色）：

```text
/world join <world_id>
```

如果你想指定在该世界里扮演的角色：

```text
/world join <world_id> <character_id>
```

进入世界后，先读一下世界卡与规则（非常重要）：

```text
/world info <world_id>
/world rules <world_id>
```

然后直接开始对话推进剧情即可（你说话，机器人会跟进）。

## 快速开始（世界创建者：发布一个可玩的世界）

### 1）打开世界创建者引导

```text
/onboard role:world creater
```

### 2）创建世界草稿并完成世界卡/规则

```text
/world create
```

你会进入一个私密编辑话题（线程）。在里面把世界卡、规则写清楚（多轮对话补全即可）。

### 3）发布世界（在编辑话题中执行）

```text
/world publish
```

发布后会创建世界子空间（频道/角色等）。玩家可以用 `/world list` 找到你的世界并加入。

### 4）处理玩家提交：提案/正典/任务/编年史

玩家或你自己可以提交内容：

```text
/world submit kind:<canon|chronicle|task|news> title:<标题> content:<内容>
```

你确认后批准写入：

```text
/world approve submission_id:<id>
```

> 原则：把“设定”写进正典（`kind:canon`），把“要做的事”写成任务（`kind:task`），把“发生的事”写进编年史（`kind:chronicle`）。

## 日常游玩常用操作（你会反复用到）

### 会话槽位：并行跑多个话题

你可以直接在消息前加 `#<key>` 来切换会话槽位（这里的 `key` 是槽位编号）：

```text
#0 继续主线剧情
#2 我想私下问一些设定问题
```

如果你发现机器人“记错了上下文”或剧情跑偏了，最直接的做法是 **重置该槽位**：

```text
/reset
/reset key:2
```

消息版同义写法：

```text
#2 /reset
```

> 槽位数量上限由群配置 `maxSessions` 决定；超过范围的槽位编号会被丢弃。

### 掷骰：不调用大模型，适合跑团裁决

```text
.rd 1d20
.rd 2d100
```

限制：

- `1 <= N <= 10`
- `1 <= M <= 100`

### 提交设定与任务：把“口嗨”变成“可追踪的世界内容”

当你想把某段内容正式写进世界（或给创作者一个可处理的提案），用：

```text
/world submit kind:canon title:某条设定 content:这里写正文
```

想提一个可执行任务，用：

```text
/world submit kind:task title:短标题 content:任务描述与验收标准
```

### 角色切换：全局默认 / 当前世界

- 全局默认角色：

```text
/character use character_id:<id>
```

- 在“当前世界”里切换你正在扮演的角色：

```text
/character act character_id:<id>
```

### 角色导入导出：迁移与备份（创作者常用）

```text
/character export
/character import file:<附件>
```

### 世界导入导出：备份世界文档（创作者常用）

```text
/world export <world_id>
/world import kind:<world_card|rules|canon> file:<附件> <world_id>
```

## 推荐的「开局模板」（复制就能用）

### 玩家开局（推荐发给机器人）

```text
我想加入一个世界开始玩。
我的偏好：
1）题材：……
2）节奏：快/慢
3）玩法：更偏剧情/更偏战斗/更偏推理
4）限制：不想出现的内容……
请给我推荐 1–3 个世界，并告诉我加入后第一幕怎么开始。
```

### 创作者开局（推荐写进世界规则）

```text
本世界的三条硬规则：
1）……
2）……
3）……

玩家在本世界“能做什么/不能做什么”：
- 能：……
- 不能：……

推进剧情的判定方式：
- 叙事优先 / 掷骰裁决 / 混合
```

## 常见问题（先看这里，能省你 80% 时间）

### 1）为什么机器人不回我？

最常见原因只有三个：

1. 你不在机器人允许对话的频道/话题里（最稳妥：先 `/onboard`，在私密引导话题里说）。
2. 你在用错会话槽位，导致上下文不对（用 `#0` 回到默认，或 `/reset` 该槽位）。
3. 你发的是斜杠指令，但 Discord 没显示候选（重新输入 `/`，确认机器人有权限、且你在支持斜杠指令的地方）。

### 2）我加入世界后该先做什么？

先看两样东西：

```text
/world info
/world rules
```

然后再开始扮演和推进剧情。别跳过规则，否则只会浪费大家时间。

### 3）我想换一种玩法/换一个模型怎么办？

- 普通用户：你能做的是换会话（`/reset`）或换槽位（`#2 ...`）。
- 管理员：可以用 `/model name:<modelId|default>` 切换群模型覆盖。
